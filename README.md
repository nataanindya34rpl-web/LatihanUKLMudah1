# Penjelasan Aplikasi
- Program meminta pengguna menginput berat, jarak, dan ukuran paket.
- Volume dihitung dari panjang × lebar × tinggi.
- Biaya per kg ditentukan berdasarkan jarak.
- Jika volume lebih dari 100 cm³, dikenakan tambahan Rp 50.000.
- Hasil total ditampilkan di akhir.
